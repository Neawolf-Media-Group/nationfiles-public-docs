# Core Hierarchy — 地政学インテリジェンス・スタックの構成

## この図の目的

**Core Hierarchy** は NationFiles システムが**役割と責任の面でどう積み上がるか**を示します — マーケ用チャートではなく**チェーン**：誰が発行し、公開面はどこにあり、分析 AI はどこで動き、指標はどこで生まれ、その下の計算層は何か。

## 各層が**すること**

| 層 | エンティティ | 製品上の役割 |
|----|--------------|--------------|
| 1 | **Neawolf Media Group** | **発行者・運営者** — インプリントと法的責任；NationFiles を資金・統括。 |
| 2 | **NationFiles** | **地政学インテリジェンスの表面** — Web、国分析、地図、ダッシュボード、製品に応じた API ビュー。 |
| 3 | **Naciro Engine** | **分析 AI** — 生シグナルを NFSI、テキスト、方法論に沿った予測コンポーネントへ圧縮。 |
| 4a | **LPU Architecture** | **推論計算基盤** — Naciro パスの決定的・高速実行。 |
| 4b | **NFSI** | **地政学的安定性／リスク・スコア** — パイプラインの定量出力（0–100）。 |

**注:** エンジンと指数は**同じ段**に並ぶ。**計算論理**と**見える結果**は別物だからです。

## この階層が**ない**もの

ネットワーク構成図でも人事組織図でもありません。**数字と分析の出自**を知りたい読者・パートナー向けの**製品ロール**の整理です。
