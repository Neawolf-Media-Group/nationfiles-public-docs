# NationFiles Stability Index (NFSI) — score de stabilité et de risque géopolitique

## Ce qu’est le NFSI

Le **NationFiles Stability Index (NFSI)** est l’**indicateur quantitatif central** de l’analyse géopolitique sur NationFiles : pour chaque pays couvert (et en agrégat mondial), il résume **stabilité versus risque documenté** dans **une note de 0 à 100**. Analystes, médias et chercheurs l’utilisent comme **repère rapide** — un « compteur » de la situation documentée, pas un rapport narratif long.

L’index est construit en **plusieurs couches** (Layers 1–3 dans la méthode) : indicateurs bruts → agrégation → score pondéré → lissage optionnel. Les entrées et pondérations figurent dans le **Validation & Verification Report** et les pages méthodologie — pour garder un score **vérifiable**.

## À quoi sert le NFSI

- **Comparaison pays :** même échelle pour tous les États couverts — classements, cartes de chaleur, séries temporelles.
- **Veille / alerte précoce :** les mouvements de score signalent dégradation ou apaisement **structurels**.
- **Langage commun :** bandes **A–E** et code couleur partagés avec les parties prenantes.

## Échelle et bandes (bref)

Les seuils suivent la logique produit (**>** à 20/40/60/80) ; détails dans la méthodologie.

## Ce que le NFSI n’est pas

Pas d’oracle pour chaque titre ; pas de substitut aux sources diplomatiques ; **pas de conseil d’investissement**. Il compresse des indicateurs documentés — pas la profondeur d’un dossier pays complet.
