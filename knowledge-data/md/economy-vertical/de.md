# Wirtschaftssparte: Makro im Kontext statt „geheimen“ Prognosemotor

Die **Wirtschaftssparte** auf NationFiles ist weniger ein neues Kennzahlen­produkt als ein **Ordnungsrahmen**: Sie sagt, wie **Einkommensvergleiche** in Kaufkraftparität, **Institutionenqualität**, **Stabilitätskontext** und **Wechselkursinformationen** auf einer Länderoberfläche **zusammen** gedacht werden sollen. Das unterscheidet sich von Werbetexten, die Krisen als lineare Kurven verkaufen.

**Fiskalpolitik** taucht dort auf, wo öffentliche Reihen es erlauben — etwa Schulden‑, Steuer‑ oder Arbeitsmarktindikatoren, die im selben Lesekontext wie das **Purchasing Power Index (PPI)**‑Bündel stehen. Wer eine Zahl vermisst, sollte zuerst prüfen, ob die **Quelle** diese Serie für das Land ausweist; ein Fehlen ist meist eine **Coverage**‑Frage, kein Indiz für Verschwiegenheit.

**Handelsstabilität** wird hier nicht mystifiziert. Gemeint ist die **sachliche** Einordnung von Reserven, Außenhandelsstruktur und **FX**‑Volatilität. Die Engine **Naciro** spielt dazu häufig aktualisierte **Wechselkurs**‑Informationen ein; sie **ersetzt** weder die dokumentierte Logik des PPI (einschließlich der Bezüge zur World Bank) noch **amtliche** Volkswirtschaftliche Gesamtrechnung.

Wenn Medien nach **Krisen** von „Erholungspfaden“ sprechen, verlangen Leser:innen zu Recht **Transparenz**. NationFiles bindet vorausschauende Formulierungen an **Refresh‑Zyklen**, **Export‑Stempel** und die Methoden, die in den jeweiligen Knowledge‑Entitäten beschrieben sind. Ein **einziges**, in diesem Graph **verborgenes** Weltmodell gibt es in den öffentlichen Daten **nicht** — das ist dokumentierte **Absicht**, keine Lücke.
