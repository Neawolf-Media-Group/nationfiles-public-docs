# Verticale économique : macro contextualisée, pas de moteur occulte

La **verticale économique** NationFiles est surtout un **cadre éditorial** : elle explique comment les **comparaisons de revenu PPA**, les **scores de gouvernance**, le **contexte de stabilité** et les **lectures croisées FX** cohabitent sur une même page pays — sans promesse de trajectoire miracles.

La **politique budgétaire** n’apparaît que lorsque des séries publiques cohérentes existent à côté du bundle **PPI** documenté dans le graphe. Une valeur absente indique le plus souvent une **limitation de couverture**, pas une opacité volontaire.

La **stabilité des échanges** consiste ici à lire réserves et ouverture **sans confondre** rangs de revenu **PPA** et volatilité **au comptant**. **Naciro** actualise le contexte de change ; il ne remplace ni la filiation publique du PPI ni l’interprétation des comptes nationaux.

Les récits de **reprise** après choc doivent rester **ancrés** dans les calendriers de publication, les horodatages d’export et les mentions prévues par le produit — pas dans un scénario macro **secret** injecté dans le jeu de données Knowledge.
