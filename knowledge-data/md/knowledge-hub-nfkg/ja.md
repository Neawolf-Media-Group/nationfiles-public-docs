# Knowledge Hub（NFKG）：未検証の巨大ノード数より「秩序」

NationFiles を**検証可能**に書くには、**ID**・**関係**・**7言語**テキストがどう揃うかを、スローガンなしで示す層が要る。**Knowledge Hub** と **NFKG** がそれだ——`/{lang}/knowledge/` 上のエンティティ・FAQ・用語集・任意 Markdown と、機械向けの **HTTPS エクスポート**。

**意味ネット**は**契約**——安定 ID、明示的 **`relations`** の辺、**fuzzy** 検索用の topics／aliases。**黙示**の法人格づくりはしない。

**CDN／エクスポート**は、公開の **`/knowledge/export/…` の HTTPS** を引用する。HTML の切り抜きは根拠にしない。**AI ガイドライン**が自動クライアントの枠になる。

**巨大なノード数**：ロードマップに出ても、**同一**数値と**射程**が**公開されたエクスポート索引メタデータ**に無ければ、NFKG 本文の**確定事実**にしない——**偽の精密さ**を避ける。
